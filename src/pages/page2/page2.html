Getting Started

This temmplate app contains all the base and plugins to getting started a project bluetooth with Ionic2 and Syrus Bluetooth Interface.

You need import in all pages that you want comunicate with syrus to libraries. this allow you send commands and receive Asycronous Response

constructor(public bt:BTCOM, public events:Events, ...)

Run this command in a page controller to set commands to Syrus via bluetooth.

this.bt.publish("command:received","QPV");

Subscribe to the specific response to get the data parsed

this.events.subscribe("command:PV", logResponse(response);
